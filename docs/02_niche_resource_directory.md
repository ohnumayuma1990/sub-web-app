# 案2：ニッチなリソースディレクトリ（リモートワーク向け地方移住支援ツールまとめ）

## 概要
地方移住を検討しているリモートワーカー向けに、自治体ごとの補助金、コワーキングスペース、お試し住宅などの情報を集約したディレクトリサイトです。

## 収益化方法
- **アフィリエイト**: 引越し見積もりサービス、不動産サイト、インターネット回線、モバイルWi-Fi、地方特産品のECサイト。
- **純広告・スポンサー**: 移住者を積極的に募集している自治体や、現地の不動産会社からの広告掲載。
- **リード獲得**: 専門の移住コンサルタントや地方求人サイトへの送客手数料。

## 維持コスト削減策
- **ヘッドレスCMSの活用**: ContentfulやMicroCMSなどの無料枠を利用してコンテンツを管理し、運用担当者がコードを触らずに更新できるようにする。
- **SSG（静的サイト生成）**: コンテンツ更新時に自動ビルド・デプロイを行う構成にし、パフォーマンス向上とサーバ負荷軽減を図る。
- **スクレイピングの自動化**: 自治体サイトなどの情報を一部自動で収集するスクリプトをGithub Actionsなどで定期実行。

## メリット・デメリット
### メリット
- 情報が集約されていること自体に価値があり、比較検討段階のユーザを囲い込みやすい。
- 自治体や企業との提携など、BtoBの収益機会がある。

### デメリット
- 情報の鮮度が重要であり、定期的なデータ更新・チェックが必要。
- 大手移住ポータルサイトとの競合。

## 想定運用費
- **ドメイン費用**: 年間 約1,500円
- **CMS費用**: 0円（MicroCMS等の無料枠）
- **サーバ費用**: 0円（Vercel等の無料枠）
- **合計**: 月額 約130円程度

## 運用負荷（人的リソース）
- **初期構築**: 2〜3週間（データ構造の設計と初期データ投入を含む）。
- **定常運用**: 週3〜5時間程度のデータ更新、情報の整合性チェック。
- **コンテンツ更新**: 月に数本の地域紹介ブログ記事作成。
